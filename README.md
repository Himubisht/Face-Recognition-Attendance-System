# Face-Recognition-Attendance-System
Face recognition attendance system using OpenCV and LBPH algorithm. This project is a real-time face recognition attendance system built using Python and OpenCV.

## Features
- Face data collection
- Model training using LBPH algorithm
- Real-time face recognition
- Automatic attendance marking

## Technologies Used
- Python
- OpenCV
- NumPy
- Pickle

## Installation

1. Clone the repository
2. Install requirements:
   pip install -r requirements.txt
3. Run training script:
   python train.py
4. Run recognition:
   python recognize.py

## Project Structure
- dataset/ → Face images
- trainer/ → Trained model
- attendance/ → Attendance records
